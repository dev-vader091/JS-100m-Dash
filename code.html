            function generateNumbers(start, end) {

              let numbers = [];
            
              for(let i = start; i <= end; i++) {
                numbers.push(i);
              }
            
              return numbers;
            }
          
        
          
                      function displayNumbers(arrayOfNumbers) {
            let tableBody = document.getElementById('results');
          
            let tableHtml = "";
          
            for(i = 0; i < arrayOfNumbers.length; i++) {
              let value = arrayOfNumbers[i];
              let className = value % 2 == 0 ? 'even' : 'odd';
          
              // if(value % 2 == 0) {
              //   className = 'even';
              // }else {
              //   className = 'odd';
              // }
          
              if(i % 5 == 0) {
                tableHtml += "<tr>";
              }
          
          
              let tableRow = `<td class=${className}>${value}/td>`;
          
              tableHtml += tableRow;
          
              if((i + 1) % 5 == 0) {
                tableHtml += "</tr>";
              }
            }
          
            tableBody.innerHTML = tableHtml;
          
          }

The code is the controller function.

getValues()

This function retrieves start and end values from the user input. These values are then parsed into numbers and then verifies with Number.isInteger() with an if() statement. If validation is successful, the two numbers are used as parameters for generateNumbers(). If not, a Sweet Alert is generated to request integers only in the case non-integer values were provided by the user.

The code is the logic function.

generateNumbers()

This function generates an array of numbers, with the first value in the array being the start value and the last the end value inputs from the user. The array generated in this function is then used as the parameter in displayNumbers().

The code is the view function.

displayNumbers()

In this function the array from generateNumbers() is used as the condition statement in a for loop. In this loop, if the index i % 5 is 0, a <tr> tag is added to the tableHtml element initialized. A tableRow variable with the <td> current i in the loop is then added to the tableHtml variable to be displayed on the page. After the loop has ended, the tableBody element from the document is assigned the tableHtml for innerHTML content, displaying the completed table to the user on the page.
